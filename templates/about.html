About the Crop Recommendation System

This application helps farmers or agricultural experts by recommending the most suitable crop for cultivation based on soil and climate conditions.

How to Use the Application

  1. Enter the values for soil nutrients: Nitrogen, Phosphorus, and Potassium.
  2. Provide additional environmental details: Temperature, Humidity, pH, and Rainfall.
  3. Click the "Get Recommendation" button to see the recommended crop.

Input Value Guidelines

Soil Nutrients

  • Nitrogen (N): Enter a value between 0 and 140 (mg/kg). This is a key nutrient for crop growth and impacts leaf and stem development.
  • Phosphorus (P): Enter a value between 0 and 80 (mg/kg). Phosphorus helps in root development and influences flowering and fruiting.
  • Potassium (K): Enter a value between 0 and 200 (mg/kg). Potassium improves water regulation, disease resistance, and overall crop quality.

Environmental Details

  • Temperature: Enter a value in Celsius (Â°C), typically ranging from 10 to 45. Ideal temperatures vary by crop, affecting growth rate and yield.
  • Humidity: Enter a percentage value (e.g., 20% to 100%). This parameter influences transpiration and water requirements of the crop.
  • pH: Enter a value between 0 and 14. Most crops grow best in a slightly acidic to neutral range, around 5.5 to 7.5.
  • Rainfall: Enter the rainfall amount in millimeters (mm), typically between 0 and 300. Adequate rainfall ensures sufficient moisture for crop growth.

Features

  • Real-time crop recommendation based on user input.
  • Displays an image of the recommended crop.
  • User-friendly form layout with helpful input guidelines.
